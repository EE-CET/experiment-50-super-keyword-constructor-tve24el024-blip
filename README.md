# Experiment 50: 'super' Keyword (Constructor)

## Problem Statement

Write a program to demonstrate the use of `super` keyword to invoke the constructor of the parent class.
Create a class `Person` with a constructor printing "Person class".
Create a subclass `Student` with a constructor printing "Student class" and calling `super()`.

## Input Format

* No input.

## Output Format

* `Person class`
* `Student class`

### Example 1

**Input:**

```text
No input
```

**Output:**

```text
Person class
Student class
```
